CSS-POSITION

        static 
            ignoriert top/right/bottom/left
        relative 
            orientiert sich bei top/left an seine ursprüngliche static-Position
            im z-index ist relative höher als static
        absolute 
            orientiert sich bei top/left an Dokument-Ursprung. 
            ist so breit wie dessen inhalt
            im z-index ist höher als relative
        fixed
            orientiert sich bei top/left an Viewport-Ursprung.
            ist so breit wie dessen inhalt
            im z-index ist höher als absolute
        sticky
            ignoriert right/bottom/left
            orientiert sich bei top an viewport
            bleibt im Elternelement
