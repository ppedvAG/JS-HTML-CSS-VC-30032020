CSS Z-INDEX

        mit z-index kann man die imaginäre Schicht in der z-Achse bei Elementen ändern
        ohne Angaben vom Web-Entwickler zu z-index werden die auto-Werte vom Browser genommen
        und sie liegen im Minus-Bereich
        Manche CSS-Eigenschaften (wie opacity oder float) lassen den z-index bei Elemente neu berechnen
