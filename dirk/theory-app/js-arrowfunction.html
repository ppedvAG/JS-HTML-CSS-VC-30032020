JS ARROWFCT

        anonyme Pfeilfunktionen lösen das Problem von `this` in JS
        (sie haben keine eigene Bindung zu diesem Schlüsselwort)

        Beachte: Ruft man die Pfeilfunktion über eine Variable auf (siehe arrow named fct),
        verhält sie sich wie die klassische Funktion
        (mit der Bindung zu `this`)
